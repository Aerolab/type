Grunt-Website

An awesome toolkit for web development

Built in Tasks

Build

                grunt

Serve

                grunt serve

Assets

Webfonts

Will create a font from SVG located in assets/webfont/*.svg

Call them using <i class="icon icon_filename"></i>

Icons

Will create a sprite from SVG&PNG. With PNG fallback

Use it using <div class="icon-filename"></div>

Image Sprites

Will create a image sprite from PNG&JPG or other formats

<div class="sprite sprite-filename"></div>

Embed SVGs

Will compress all SVG in one simple Base64 CSS

It will be embeded with JS, enabling stylign and animation

<div class="illus-filename" data-grunticon-embed>

Assets

Any other assets inside /assets/ will be available

Bower

Install any requirement with Bower.js

bower install jquery --save

SASS

SASS is quickly compiled with libsass

Templates

Works like a charm with Handlebars
